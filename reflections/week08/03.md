## Questions

1. What is Software Development Testing?

It is writing code that allows you to test the code of your project without having to start up the app, login, set up all the conditions in the app you want to test over and over.

2. What are the benefits of Testing in software development?

It saves a lot of time. Creating the conditions in the app over an over eats up most of the time. You are able to make sure that each piece of a larger system is working. The tests could be recyclable into other projects.

3. What are some potential drawbacks of Testing in software development?

It takes more upfront work. And it is hard to predict that the time cost to set up the testing will actually be worth it.
